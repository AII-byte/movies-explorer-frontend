# movies-explorer-frontend

"Состояние" меню не реализовано полноценно, тк это уже добавление функциональности: регистрация и логин пользователя, но все "состояния" свёрстаны.
Чтобы проверить функциональность burger menu (как свёрстан header для залогиненного пользователя), необходимо в компоненте App.js закоммитить код, включая ``` <nav></nav> ``` и раскоммитить ``` <BurgerMenu /> ```.
